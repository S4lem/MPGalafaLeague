DESCRIPTIF:
Ce projet est qu'un simple exemple d'utilisation de la base Mongodb avec C#
Afin que tous puissent comprendre comment cela fonctionne
--

OU TROUVER LE CODE C#
le fichier de code se trouve dans le repertoire suivant: MPGalafaLeague\LibraryApplication\LibraryApplication
Il s'appel Program.cs
--

COMMENT LANCER LE PROGRAMME
Le mieux est de double cliquer sur le fichier LibraryApplication.sln dans MPGalafaLeague\LibraryApplication
Afin qu'il vous ouvre le programme directement avec toutes les libraries intègrées. 

Avant de Run le programme, veuillez lancer "Mongod" afin de permettre à l'utilisateur d'ouvrir la BDD Mongo
puis lancez le programme.
--

RESULTATS
Vous allez remarquer que il créer une nouvelle collection
Dans cette collection vous trouverez les éléments qui ont été ajouté dans le code

Le code vous retourne deux requêtes sur la nouvelle collection créee.

Testé sous visual Studio 2015 fonctionne correctement RAS.
